# 🜍 Nemos Alchemie

*Ein alchemistisches Browsergame — sammle Symbole, entdecke Rezepte, werde ein Meister der Alchemie.*

## Spielanleitung

Öffne `index.html` in einem beliebigen Browser — kein Server nötig.

### So spielst du

**Symbole sammeln**
- Klicke auf die schwebenden Symbole in der Welt
- Gesammelte Symbole erscheinen in deinem Grimoire (📖)

**Rezepte kombinieren**
1. Wähle ein Symbol aus deinem Grimoire
2. Klicke auf ein zweites Symbol
3. Wenn ein gültiges Rezept existiert → neues Element enthüllt!

**Die Rezepte (probieren!)**
- 🜂 Ignis + 🜄 Aqua = 🜈 Vita
- 🜃 Terra + 🜄 Aqua = 🜂 Ignis
- 🜁 Aether + 🜂 Ignis = 🜃 Terra
- 🜈 Vita + 🜃 Terra = 🜁 Aether

**Level-System**
- Gesammeltes Symbol: +2 XP
- Neues Rezept: +5 XP
- Level aufsteigen = besserer Alchemist

**Steuerung**
- Linksklick: Sammeln / Auswählen / Kombinieren
- 📖 Taste: Grimoire öffnen/schliessen

## Technische Details

- **Format:** Single HTML File (kein Server, keine Dependencies)
- **Grösse:** ~11 KB
- **Speicher:** LocalStorage (Spielstand bleibt erhalten)
- **Sprache:** Deutsch
