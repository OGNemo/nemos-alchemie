# 🜍 Nemos Alchemie — Anleitung

## Spiel starten
Öffne `projects/game/mvp/index.html` im Browser.

## Wie man spielt

### Symbole sammeln
- Klicke auf Symbole in der Welt um sie zu SAMMELN
- Gesammelte Symbole leuchten und erscheinen im Grimoire

### Rezepte kombinieren
1. Wähle ein GESAMMELTES Symbol (es leuchtet)
2. Klicke auf ein zweites gesammeltes Symbol
3. Wenn ein Rezept existiert → neues Element freigeschaltet!

### Die 4 Rezepte (probieren!)
- 🜂 Ignis + 🜄 Aqua = 🜈 Vita (Dampf des Lebens)
- 🜃 Terra + 🜄 Aqua = 🜂 Ignis (Schlamm entzündet sich)
- 🜁 Aether + 🜂 Ignis = 🜃 Terra (Geist manifestiert sich)
- 🜈 Vita + 🜃 Terra = 🜁 Aether (Leben erblüht)

### Level-System
- Jedes gesammelte Symbol: +2 XP
- Jedes neue Rezept: +5 XP
- Level aufsteigen = schnelleres Freischalten

## Steuerung
- **Linksklick** auf Symbol zum Sammeln/Auswählen/Kombinieren
- **📖 Grimoire** Taste oben rechts = alle entdeckten Symbole

## Save
- Automatisch in LocalStorage gespeichert
- Page refresh = Spielstand bleibt
