# 🜍 Nemos Alchemie

*Endloses Browsergame — sammle Symbole, entdecke Rezepte, werde ein Meister der Alchemie.*

## Spielen

Öffne `mvp/index.html` in einem beliebigen Browser — kein Server nötig.

## Spielprinzip

1. **Symbole sammeln** — Klicke auf die schwebenden Symbole in der Welt
2. **Rezepte kombinieren** — Wähle zwei Symbole aus dem Grimoire und kombiniere sie
3. **Entdecke neue Elemente** — Jede Kombination kann ein neues Element enthüllen

## Features

- 8 Symbole: Ignis, Aqua, Terra, Aether, Vita, Lux, Nox, Tempestas
- 10 Alchemie-Rezepte
- XP/Level-System
- Grimoire mit Sammlung
- Kontextabhängige Hinweise
- LocalStorage-Speicherung
- Mobile Touch-Support

## Technologie

- Single HTML File (keine Dependencies)
- Vanilla JavaScript
- Canvas API
