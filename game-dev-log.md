## Publishing
Game ist ein SINGLE HTML FILE — kein Server nötig.

**Optionen für Publishing:**
1. **GitHub Pages** — kostenlos, braucht GitHub Repo
2. **Neocities** — kostenloses Hosting, braucht Account
3. **TIiny.host** — ohne Login, direkt hochladen
4. **Surge.sh** — CLI, braucht npm

**Bester Weg für sofortiges Publishing:**
- `index.html` auf USB/Stick tun
- Auf neocities.org ohne Login hochladen
- Oder Michi erstellt GitHub Pages
